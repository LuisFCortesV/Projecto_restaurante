# Administrador de restaurante - C++

Programa de consola en C++ para la administración básica de un restaurante. Permite cargar ingredientes y platos desde
   archivos de texto, calcular precios automáticamente aplicando IVA (19%), registrar órdenes validando inventario
  disponible, y consultar los platos más solicitados y más rentables.

  **Archivos de datos requeridos: ingredientes.txt, platos.txt
